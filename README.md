# Online Store - React + Redux Ecommerce App

A modern ecommerce web application built using React, Redux Toolkit, and React Router.  
Users can browse products, view product details, add items to cart, and checkout.

---

##  Features

-  Home Page with product listing
-  Product Details page
-  Add to Cart functionality
-  Increase / Decrease Quantity
-  Automatic Total Price Calculation
-  Dollar to Rupee (INR) Conversion
-  Remove from Cart
-  404 Not Found Page
-  Dynamic Routing
-  Fast UI using Vite

---

## Tech Stack

- React
- Redux Toolkit
- React Router DOM
- Vite
- CSS
